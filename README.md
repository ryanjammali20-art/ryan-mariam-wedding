# Ryan & Mariam Wedding Website

Bilingual wedding website for guests, with French and English content and RSVP links pointing to Joy.

## Current setup

- `index.html` is the full website.
- French is the default language.
- Guests can switch between French and English.
- RSVP buttons link to Joy:
  `https://withjoy.com/ryan-and-mariam-oct-26?utm_medium=web&utm_source=joy&utm_campaign=website_overview_copy`
